Git : Commandes de base

CREER UN DOSSIER

mkdir monPremierRepo

EN FAIRE UN REPO GIT

git init

AJOUTER UN FICHIER

git add nomDeVotreFichier.extension

AJOUTER TOUS LES FICHIERS DU DOSSIER

git add .

FAIRE UN COMMIT

git commit -m "Description du commit"

VOIR L'HISTORIQUE DU COMMIT

git log

RETROUVER UN COMMIT

git checkout SHADuCommit

CLONER UN REPO GITHUB

git clone lienFourniParGitHub

RECUPERER EN LOCAL LES MODIFS FAITES SUR GITHUB (branche master)

git pull origin master

ENVOYER LES FICHIERS MODIFIES SUR SON REPO GITHUB (branche master)

git push origin master
